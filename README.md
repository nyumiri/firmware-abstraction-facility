# Firmware Abstraction Facility (FAF)

Uma biblioteca modular para controle de hardware, focada em arquitetura profissional e baixo acoplamento

> **Status:** Em refatoração (Branch `Dev` Ativa).

## Arquitetura
Esta biblioteca utiliza um design modular inspirado em frameworks de baixo nível, permitindo que o usuário inclua apenas o necessário para sua aplicação.
